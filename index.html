Block quotations

> this is a block quote
> made of many lines

This lazy way to write it gives the same result

>  this is a block quote
   made of many lines

Verbatim (code) blocks

Use 4 spaces or a tab to include a block text as it is.

    > this is a block quote  
      made of many lines

Begin with a row of three or more tildes (~) and you get a fenced code block.

~~~~~~~
> this is a block quote  
  made of many lines
~~~~~~~

To highlight specific code formatting use:

```r
x <- 1:10;
print("hello");
```

this is the result

x <- 1:10;
print("hello");
